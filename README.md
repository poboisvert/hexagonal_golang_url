# hexagonal_golang_url

go mod init github.com/poboisvert/hexagonal_golang_url

## Postman POST

```
{
    "url": "https://github.com/teris-io/shortid"
}
```

## Postman GET

Response

```
{
    "code": "GfUMZZMVR",
    "url": "https://github.com/teris-io/shortid",
    "created_at": 1662163762
}
```

Use in URL

http://localhost:8000/GfUMZZMVR
